# 宇宙重力场模拟器

一个离线可运行的交互式宇宙重力场模拟器。你可以在画布中手动放置黑洞、脉冲星、恒星、巨行星、行星和彗星，拖拽设置初速度，并观察多体引力下的轨道演化、轨迹、重力场方向、时空网格扭曲和近似相对论效应。

## 功能

- 手动放置天体，并用拖拽矢量设置初速度
- 支持自动估算圆轨道初速度
- 支持黑洞、脉冲星、恒星、巨行星、行星、彗星
- 支持轨迹、重力场箭头、时空网格和引力透镜式视觉效果
- 支持牛顿多体引力和弱场后牛顿近似相对论修正
- 支持碰撞合并、黑洞吞噬、导入/导出模拟状态
- 内置预设：微型太阳系、完整太阳系、双星交换、黑洞近掠、脉冲星伴星
- 单文件应用，无需安装依赖

## 运行

直接用浏览器打开 `index.html` 即可。

也可以用任意静态文件服务器运行，例如：

```bash
python -m http.server 8000
```

然后访问：

```text
http://localhost:8000
```

## 上传到 GitHub

1. 新建一个 GitHub 仓库。
2. 将本目录下的所有文件上传到仓库根目录。
3. 如果要启用 GitHub Pages：
   - 进入仓库 `Settings`
   - 打开 `Pages`
   - Source 选择 `Deploy from a branch`
   - Branch 选择 `main`，目录选择 `/root`
   - 保存后等待 GitHub 生成访问地址

## 物理模型说明

模拟核心使用二维 N-body 多体引力积分。相对论效果采用弱场/后牛顿近似，用于表现近大质量天体时的轨道进动、强引力捕获和视觉上的时空扭曲。它不是完整的广义相对论数值求解器，也不会求解爱因斯坦场方程。

“完整太阳系”预设是适合实时交互的压缩比例示意模型，保留行星顺序、质量层级和视觉特征，但不是按真实星历、真实距离和真实尺寸比例构建。
